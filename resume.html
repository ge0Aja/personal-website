PROJECTS

EIT - Augmented Retail (AR) — Retail Store Assistant App

AR is a European project under EIT Digital that aims to use Augmented Reality and Computer Vision  to develop an application that assists employees in retail supermarkets with their daily tasks. My work is related to the computer vision engine which detects and classifies retail store products and tags using an ensemble of Deep Learning and classical Computer Vision techniques.

EIT - Approxim@action — IoT Data Protection

Approxim@ction is an EIT Digital European project that aims to deploy a smart Information prediction Algorithm to be used in Low Power WANs and IoT networks to monitor different aspects of the atmosphere. My work is related to designing and implementing a “Data Protection mechanism” for safe key exchange and efficient data encryption.

CrowdApp — CrowdSourcing for Mobile Apps Rating

CrowdApp is a mobile application that helps in crowdsourcing usage information and alerts about mobile apps installed by a crowd on a network. The brain engine of CrowdApp uses Game Theory to rank these apps. My work is related to implementing the mobile tool on Android & a backend server running PhP & MySQL server.
