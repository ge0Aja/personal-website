My Thesis

As a partial requirement to graduate from the master's program I wrote my thesis about Mobile Network Classification - See full print here.
I tried to provide a solution using a classification engine based on ML algorithms to classify and detect abnormal network traffic generated by mobile applications. The classification engine used supervised and semi-supervised ML classification models to detect abnormal traffic. The main contributions of the thesis are:
  • Vector of network features used to feed the model
  • Comparison between different classification models
  • Formulation of a similarity score used in a Bayesian probability algorithm to differeniate traffic patterns
